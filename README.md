# 기술 스택 배지 대시보드

본 프로젝트는 Flask를 활용하여 개인의 기술 스택을 동적으로 관리하고 출력하는 웹 애플리케이션입니다.

## 주요 기능
- **입력 기능**: 새로운 기술명을 입력하여 리스트에 추가
- **출력 기능**: 저장된 리스트 데이터를 화면에 반복 출력 (Jinja2 for문)
- **조건부 출력**: 특정 기술명(Python, SQL 등)에 따른 자동 배지 스타일 적용
- **예외 처리**: 빈 값 입력 시 리스트 저장 방지

## 실행 방법
1. 가상환경 활성화: `source .venv/bin/activate`
2. 필수 패키지 설치: `pip install -r requirements.txt`
3. 서버 실행: `python3 app.py`
