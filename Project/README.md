# 수행평가
## 1. 주요 기능
- **입력 기능**: 새로운 기술명을 입력하여 리스트에 추가
- **출력 기능**: 저장된 리스트 데이터를 화면에 반복 출력 (for문)
- **조건부 출력**: 특정 기술명(Python, SQL 등)에 따른 자동 배지 스타일 적용
- **예외 처리**: 빈 값 입력 시 리스트 저장 방지
## 2. 사용한 문법
- **index**
- **loop.index0**
- **form**
- **hidden input**
- **request.form**
- **render_template**
- **redirect**
- **if / elif / else**
- **for**
.strip()
.append()
## 3. 실행 결과
- 입력창에 기술명을 입력하면 실시간으로 목록에 데이터가 추가됨
- 조건문(if)을 통해 특정 기술명(Python, SQL 등)에 맞는 전용 배지 스타일이 적용됨
- 빈 값을 입력할 경우 리스트에 저장되지 않는 예외 처리가 작동함
- Redirect 기능을 통해 데이터 전송 후 메인 화면에서 결과를 바로 확인할 수 있음
